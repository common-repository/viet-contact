== Installation ==

=== From within WordPress ===

1. Visit 'Plugins > Add New'
2. Search for 'Viet contact'
3. Activate Viet contact from your Plugins page.

=== Manually ===

1. Upload the `Viet-contact` folder to the `/wp-content/plugins/` directory
2. Activate the Viet contact plugin through the 'Plugins' menu in WordPress
3. Go to "after activation" below.
